ETPL

Enterprise Template是一个灵活、具有强大复用能力的高性能模板引擎
适用于WEB前端应用中视图的生成，特别是SPA(Single Page APP)类型的应用

Start

ETPL可以在CommonJS/AMD的模块定义环境中使用，也能直接在页面下通过script src=的方式引用：

  1. CommonJS/AMD环境下需要通过如下代码得到ETPL的模块。
    var etpl = require( 'etpl' );
  2. 得到ETPL模块对象后，首先对模板源代码进行编译，会能得到编译后的function。
    var render = etpl.compile( 'Hello ${name}!' );
  3. 然后执行这个function，传入数据对象，就能得到模板执行的结果了。
    var text = render( {name: 'etpl'} );
